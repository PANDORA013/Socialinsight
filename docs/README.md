## 📊 System Overview

**Platform:** Web-based Social Media Analytics  
**Framework:** Laravel 11 + Python 3.13.9  
**Database:** MySQL 8.0  
**AI Models:** Naive Bayes + IndoBERT (99% accuracy)

### Supported Platforms

- ✅ YouTube (Videos & Comments)
- ✅ Twitter (Tweets)
- ✅ TikTok (Videos)
- ✅ Instagram (Posts)

### Key Features

- 🤖 AI-powered sentiment analysis (99% accuracy)
- 📊 Topic clustering with K-Means
- 🎯 Actionable recommendations
- 📈 Real-time trend detection
- 🔍 Advanced data filtering
